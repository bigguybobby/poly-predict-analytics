# Poly Predict Analytics

Prediction market analytics and tooling platform for the Polygon ecosystem (Polymarket integration). Built for the Polygon Prediction Markets Program.

## Features
- Real-time Polymarket data aggregation
- Market sentiment analysis
- Historical accuracy tracking
- Portfolio P&L dashboard
- API for prediction market data

## Tech Stack
- **Frontend:** Next.js, TypeScript, Tailwind CSS, wagmi/viem
- **Chain:** Polygon PoS
- **Data:** Polymarket APIs, The Graph
